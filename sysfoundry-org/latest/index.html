Welcome to Sysfoundry Documentation

You are in the Sysfoundry Documentation site. Here you will find details about the Sysfoundry project.

Overview

Sysfoundry is a rapid application development platform built on top of the Spring Platform and the Apache Isis projects. The main focus of Sysfoundry is to enable application developers to conceptualize and rapidly develop domain specific applications.

Predominantly Sysfoundry adopts the Naked Objects framework methodology and is biased towards the Domain Driven Design paradigm of solving business domain problems. You can find more details about Naked Objects here.
